Для того чтобы начать работать с данной сборкой нужно:
Установить gulp  и nodejs через npm глобально;
Скопировать данные файлы себе на проект;
Открыть интегрированный терминал внутри проекта;
Далее поочередно ввести команды:
- npm init;
- npm install gulp --sev-dev;
- gulp (тут откроется локальный сервер, и начнут работать все таски)
Команду gulp нужно вводить каждый раз когда открываешь проект
